  2. HTML est un langage de balisage, on y retrouve tout ce qu'on perçoit sur un site internet: Le menu, l'interface, le bas de page, le haut de page, tout ce qu'on retrouve sur un site internet. C'est la base de la création d'un site. Le HTML permet de structurer la page c'est dedans que'on retrouve le contenu comme les textes et les images...
  3. Qu'est ce que vous en déduisez et ce que vous comprenez de son fonctionnement?
  4. On comprend que le HTML est la base la plus important d'une céation, créé du contenu que l'on retrouve sur la page. HTML est un langage qui contient des balises, ces balises sont utilises pour organiser la structure d'une page internet. Certaines balises permettent d'intégrer certains médias comme images vidéos textes... C'est le navigateur qui interprète ce langage au même titre que le javascript et le css, tous les navigateurs n'interprètent pas les mêmes balises de la même manière.
